# React Dashboard 模板

基于 React + TypeScript + Tailwind CSS 的管理后台模板。

---

## 技术栈

- **框架**: React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **组件库**: Headless UI / Radix UI
- **状态管理**: Zustand / React Query
- **路由**: React Router v6
- **构建**: Vite
- **图表**: Recharts

---

## 适用场景

- 管理后台系统
- 数据可视化面板
- CRM/ERP 前端
- 配置管理界面

---
